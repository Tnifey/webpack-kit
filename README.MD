# Webpack Kits
Startup bundles with basic configured webpack

## Bundles

### Basic Bundle
``` bash
git clone -b basic https://github.com/Tnifey/webpack-kit.git
```

#### What contains?
*   webpack  
*   browser-sync-webpack-plugin
*   html-webpack-plugin
*   html-loader  
*   css-loader
*   style-loader
*   babel-core (preset-env)
